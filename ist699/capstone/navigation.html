Navigation Help

Use this page if you are having trouble opening modules, seeing content, or moving through the course. The modules are created in iSpring and posted on GitHub as web pages.

1. I clicked a module, but nothing seems to happen.
  • Wait a few moments to allow the page to load, especially on slower connections.
  • Try selecting the link again once.
  • Check whether the module opened in a different tab or window in your browser.
  • If pop-up blocking is enabled, allow the site if your browser prompts you.
If the link still does not open, try a different browser (for example, Chrome, Edge, or Firefox).
2. The page loads, but the module player looks blank or partially loaded.
  • Refresh the page once using your browser’s refresh/reload button.
  • Confirm that your internet connection is active and stable.
  • Check whether any browser extensions (for example, script-blocking tools) might be preventing content from loading.
  • If you are on a school or district network, some content may load more reliably in a different browser.
3. I cannot hear any audio.
  • Check your device volume and confirm that your speakers or headphones are connected.
  • Look for a mute/unmute or volume control within the module player itself.
  • Make sure the browser tab is not muted.
  • If you are using a headset, try disconnecting and reconnecting it, then reload the page.
4. The module controls (Next, Previous, Play/Pause) are hard to find.
  • Most iSpring modules include controls near the bottom or edges of the player.
  • Look for arrows or “Next/Previous” labels to move between slides or sections.
  • Some modules also show a list or timeline where you can see your current position.
If you still cannot locate the controls, try resizing your browser window or switching to full-screen mode, if available in the player.
5. The module is taking a long time to load.
  • Allow extra time for the first load, especially if the module includes media-rich content.
  • Close other tabs or applications that may be using your internet connection.
  • If possible, move to a stronger connection (for example, a wired connection or closer to your router).
6. I’m not sure where I am in the course.
  • Return to the Modules page to see all modules at a glance.
  • Identify the last module you completed or partially completed.
  • Use the estimated times and descriptions to choose where to resume.
7. I want to review a specific idea again.
  • Open the module that introduced that idea (for example, UDL principles in Module 2 or lesson design in Module 3).
  • Use the player controls to move quickly to the relevant section.
  • Pause when you reach the content you want to review and take any additional notes.

Navigation at a Glance

Use this as a quick guide to move around the course site and modules.

From GitHub course site
  • Home: High-level course description and purpose.
  • Modules: Cards and links to the Course Overview and Modules 1–6.
  • How-to Guide: Suggested pacing and strategies for working through the course.
  • Navigation Help: This page, for troubleshooting.
Within an iSpring module
  • Use Next and Previous to move between sections.
  • Look for a progress indicator to see how far you’ve gone.
  • Pause any time you want more time on a slide or activity.
If you get stuck
  • Refresh the page once.
  • Try a different browser.
  • Return to the Modules page and reopen the module.

For most issues, returning to the Modules page and reopening the desired module will help you get oriented again. You can use the How-to Guide for pacing and usage tips.
